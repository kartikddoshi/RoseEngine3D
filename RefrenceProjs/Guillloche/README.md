# Digital Rose Engine Simulator

This project is a software tool to simulate the behavior of a rose engine, generate decorative patterns, and create G-code for CNC machining.

## Features

-   Parametric modeling of rose engine motion
-   Live vector-based preview of patterns
-   G-code generation for CNC machines

## Setup

1.  Clone the repository.
2.  Install dependencies: `pip install -r requirements.txt`
3.  Run the application: `python rose_engine_simulator/main.py`
